# GraphQL CEP 

Query any CEP (Brasil) using GraphQL

Test it here: https://graphql-cep-liimjftpnr.now.sh

![graphql-cep](screenshot/graphql-cep.png)

## Command

#### Setup
```bash
npm install
```
#### Develop
```bash
npm run watch
```

#### Production
```bash
# first compile the code
npm run build

# run graphql compiled server
npm start
```

### Flow
```bash
npm run flow
```

Or
```bash
flow
```
